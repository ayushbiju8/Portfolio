## ✨ Features
- **🔥 Brutalist Aesthetic:** Bold, unapologetic design with stark contrasts (deep reds, blacks, and whites) and rigid grid structures.
- **⚡ Buttery Smooth Scrolling:** Powered by **Lenis**, the scrolling experience is silky smooth, hijacking the native scroll for a cinematic feel.
- **🎬 Advanced Animations:** Heavy use of **GSAP ScrollTrigger** for staggered drop-ins, reveal lines, ambient glowing, and continuous marquees.
- **🕹️ Interactive Physics:** An interactive "Experience" section where a red paper notice responds to a toggle switch—complete with a rotating fan!
- **📬 Working Contact Form:** Fully integrated with **Web3Forms** so visitors can shoot me an email directly without leaving the page.
- **📱 Fully Responsive:** Carefully crafted to ensure the complex grid layers and 3D effects scale down perfectly to mobile screens.

## 🛠️ Tech Stack
- **React.js** (via **Vite**) - For a lightning-fast development experience and optimized build.
- **Tailwind CSS v4** - For rapid utility-first styling and precise structural layouts.
- **GSAP (GreenSock)** - The backbone of all scroll-reveals and complex timeline animations.
- **Lenis** - For providing that premium, uninterrupted smooth-scroll experience.
- **Web3Forms** - Handling backend form delivery seamlessly.

## 💻 Running it Locally

If you want to pull this down and poke around the code, it's super easy to get running on your machine:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/ayushbiju8/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Spin up the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` (or whatever port Vite gives you!).
